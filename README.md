Для локального запуска нужно выполнить следующие команды:
```
git clone https://github.com/AlexandrKananadze/java-project-lvl4.git SeoAnalyzer
cd SeoAnalyzer
make start
```
В результате чего, проект запустится по адресу `http://localhost:4000`
